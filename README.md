# SQL Joins and Built-in Functions

This repository contains intermediate SQL practice using a school database to demonstrate joins and commonly used built-in functions.

## Project Overview
The goal of this repository is to strengthen SQL skills needed for reporting, text transformation, date calculations, and combining related tables.

## Topics Covered
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- ROUND
- ABS
- MOD
- CONCAT
- LENGTH
- REPLACE
- SUBSTRING
- UPPER
- LOWER
- NOW
- DATEDIFF
- DATE_ADD

## Database Used
StudentsDB

## Files Included
- `01_problem_statements/`
- `02_schema_and_setup/`
- `03_solutions/`

## Key Learning Outcomes
- Joined multiple related tables correctly
- Applied string functions for text formatting
- Used numeric functions for data validation examples
- Used date functions for timeline and follow-up calculations
